EMPDateFormatter

@interface EMPDateFormatter : NSObject

Use this little helper for parsing and formatting dates for the Emplate API

The format used in this class is: yyyy-MM-dd'T'HH:mm:ssZZZ which is formatted like: 2017-05-31T13:37:42+0000.
