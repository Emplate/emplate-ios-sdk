UIDevice(EMPBeaconSupport)

@interface UIDevice (EMPBeaconSupport)

This is a helper category for the UIDevice class. Use the properties in this category to determine if the phone is ready to search for beacons.

  • bluetoothEnabled

    Boolean to indicate wheter bluetooth is enabled or not

    Declaration

    Objective-C

    @property (readonly, atomic) BOOL bluetoothEnabled;
  • hasAuthorizedLocationAlways

    Boolean to indicate wheter the user has authorized location usage always

    Permissions to location usage always is needed to search for beacons in background.

    Declaration

    Objective-C

    @property (readonly, atomic) BOOL hasAuthorizedLocationAlways;
  • supportsBeacons

    Boolean to indicate wheter the device is supporting beacons

    The simulator, iPhone 4 and older device does not support beacons.

    Declaration

    Objective-C

    @property (readonly, atomic) BOOL supportsBeacons;
